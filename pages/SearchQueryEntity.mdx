# Query Entity

This endpoint returns a JSON object containing information about Entites queried from the Rahn Monitor API.


```
https://api.rahnmonitor.co.za/searchqueryentity
```

## Path Parameters

| Parameter |  Type  | Description                                                                                            | Required? | Default |
| :-------: | :----: | ------------------------------------------------------------------------------------------------------ | :-------: | :-----: |
|  apikey     | string | API Key for RahnMonitor |    Yes    |         |
|  EntityID | string | Entity ID consists of a hash in the Rahn Monitor Database System |    No    |         |
|  EntityName | string | Entity Name|    No    |         |
|  City | string | City|    No    |         |
|  State | string | State|    No    |         |
|  Country | string | Country|    No    |         |
