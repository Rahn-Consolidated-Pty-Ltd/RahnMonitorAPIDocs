# Check Balance

This endpoint returns a JSON object containing the top 5 local adverse media sanctions articles for today.


```
https://api.rahnmonitor.co.za/ReturnLocalSanctionsTop5Today
```

## Path Parameters

| Parameter |  Type  | Description                                                                                            | Required? | Default |
| :-------: | :----: | ------------------------------------------------------------------------------------------------------ | :-------: | :-----: |
|  apikey     | string | API Key for RahnMonitor |    Yes    |         |


## Request Samples

<Tabs items={["JavaScript", "Python","Curl"]}>
  <Tab>
    <>
        ```js
        import axios from "axios";
        
        const url = "https://api.rahnmonitor.co.za/ReturnLocalSanctionsTop5Today?apikey=MYAPIKEY";
        const data = async () => {
            try {
                const { data } = await axios.get(url, 
                    {
                        headers: {
                            "Content-Type": "application/json",
                        },
                    }
                );
                return data;
            } catch (err) {
                throw new Error(err.message);
            }
        };

        console.log(data);
        ```
    </>
